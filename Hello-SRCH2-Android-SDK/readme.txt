Hello.

This is the demonstration project for the SRCH2 Android SDK.

This project contains two modules:
  -module 'demo-basic' contains the Hello SRCH2 Android SDK demonstrating use of the SRCH2 Android SDK
                    as it explained on the SRCH2 Android SDK Documentation - Basic webpage
  -module 'demo-sqlite' contains example usage of the SQLiteIndexable class for connection the SRCH2
                    Android SDK to a pre-existing SQLite database table as it is explained in
                    the SRCH2 Android SDK Documentation - Advanced webpage

Search on!
